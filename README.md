## ğŸ¯ Objective
**``Questions``** is a project designed to help teachers **``create and grade exams automatically``**.
It **``facilitates the creation of exams by reusing different questions``** that can be stored in a database, while also allowing users to create new questions when needed.
The system also performs grading through a unique serial number generated to identify each exam, which is attached using a **``QR code``** that can be scanned to automatically grade the test.
